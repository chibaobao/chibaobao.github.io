web.py使用

post和get请求

  • 主要部分

    1. urls
    2. 与urls中映射相同的类名，定义GET或POST函数相应相关请求
    3. app = web.application(urls, globals())
    4. app.run()

  • web.input()

    • 表示url中的参数(get,post)
    • eg:http://127.0.0.1:8080/?name=aa&id=123
  • web.data()
    • 表示上传的数据
    • eg:http://127.0.0.1:8080/?name=aa&id=哈哈
      • body内容：{ “name”: “aa”, “id”: “bb”}
  • json.dumps(dict)
    • 将字典转换为json
